Stellar Tales
=============

Stellar Tales is a massive multiplayer space simulation game. It uses only text for visual effects so it's highly dependent on players imagination.


Tech behind the game
====================
- JavaScript
- Node.js
- Express
- Socket.IO
- Hosted on RaspberryPi
